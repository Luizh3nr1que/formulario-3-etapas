# formulario-3-etapas

<b>Este projeto consiste em um formulário de 3 etapas implementado utilizando JavaScript, HTML e CSS. O formulário é responsivo, exibe mensagens de erro quando necessário e fornece uma mensagem de conclusão ao finalizar o preenchimento.<b>
